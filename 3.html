서버 공부

프론트엔드의 개념 완성 후엔 서버를 관리하는 백엔드 공부를 시작합니다. 고객들에게 서비스를 제공 해준다는 뜻을 가진 서버란 개발에서는 사용자 눈에 보이지 않는 정보를 말합니다.
백엔드는 원래 공부 할 양이 방대하지만 요즘은 프레임워크의 도움을 받아 비교적 쉽게 만들 수 있습니다. Javascript, Ruby, Python 등 쉬운 언어 중 하나를 선택합니다. 쉽다고 해도 처음부터 공부하는 것은 시간이 오래 걸립니다. 우선 기본개념인 변수, 조건문, 반복문, 함수 4가지 개념을 먼저 학습합니다. 그 후 더 필요한 것은 그 부분만 찾아가면서 공부하면 됩니다.
개념 학습이 끝 난 후에는 프레임워크의 사용법을 익혀봅시다. 인프런, 노마드코더 같은 학습 사이트에서 클론코딩을 수강하면서 익히면 됩니다. 각 언어의 프레임워크로는 Javascript spring, Python Django, Ruby Rails등이 있습니다.
프레임워크를 학습하면서 Wed Client 학습하며 만들어 둔 웹에 적용해봅시다.
